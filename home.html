XSCRIPT

New, Fresh, Simple


Summary

XSscript is a coding language created in C++ using LLVM. We want to make a kind of hybrid between Python and C++. We want the comprehensive syntax of Python and the speed of C++.


Goals

Our goal with this coding language is to be begginer friendly, modular, fast and open source.


Developemet

We've been working on this new coding language for some time.
